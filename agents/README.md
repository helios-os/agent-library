# Helios Agent Library

This directory contains operational AI agents developed as part of the Helios OS ecosystem.

Each agent focuses on a single operational task.

Principle:

One problem → one agent.

## Current agents

### Receipt Agent
Automates receipt processing and prepares structured data for bookkeeping.

### Visibility Agent
Helps businesses become understandable and recommendable for AI search systems.

### Booking Agent
Handles reservation requests and availability workflows.
